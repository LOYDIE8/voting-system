======================================================
CTU-DB ONLINE VOTING SYSTEM - ADMIN GUIDE
======================================================

HOW TO LOG IN AS ADMIN:
1. Open the voting system in your web browser.
2. Under the student login box, click on "Admin Access".
3. Enter the admin credentials:
   - Username: admin
   - Password: admin123
4. Click the "Access Dashboard" button.

HOW TO RESET ALL VOTES TO ZERO:
1. Log in to the Admin Dashboard using the steps above.
2. Click the red "Reset Entire Election" button.
3. A warning pop-up will appear asking if you are sure. Click "OK" to confirm.
4. An alert will notify you that the system reset is complete and all votes are back to 0.
